## SYNOPSIS

```sh
cargo version [options]
```

##### DESCRIPTION

Displays the version of Cargo.

## OPTIONS

```sh
-v
--verbose
```

Display additional version information.

##### EXAMPLES

Display the version:

```sh
cargo version
```
The version is also available via flags:

```sh
cargo --version
cargo -V
```
Display extra version information:
```sh
cargo -Vv
```
